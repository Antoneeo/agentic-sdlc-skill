# Architettura del Progetto

## Stack Tecnologico
- **Linguaggio:** JavaScript (Node.js)
- **Tooling:** Gemini CLI, npm
- **Skill Engine:** Gemini Skill Framework (Custom "agentic-sdlc")

## Struttura delle Directory
- `agentic-sdlc/`: Sviluppo della skill personalizzata.
- `docs/`: Documentazione tecnica (Documentation-First).
- `docs/solutions/`: Documenti di analisi per le singole feature.

## Pattern Architetturali
- **Documentation-First SDLC**: Ogni feature deve essere documentata prima dell'implementazione.
- **Pattern AAA per i Test**: Arrange, Act, Assert.
