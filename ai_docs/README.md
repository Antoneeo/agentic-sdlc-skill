# ai_docs — reading guide

Must-reads for this repository, in order. The full manifest of canonical docs is
`INDEX.md` (generated — regenerate with `sdlc_check.py index`, never edit by hand).

1. `vision/project_vision.md` — why the skill exists + the four-layer product thesis (check its Status first).
2. `strategic/capabilities_and_positioning.md` — what agentic-sdlc already does better, parity, and positioning vs superpowers.
3. `vision/roadmap_evoluzione_agenti.md` — the active evolution roadmap (subagents, operative guides, devPNT seam).
4. `strategic/architecture.md` — how the package is built.
5. `audit/handoff.md` — where work stopped last session (if present).

Directory purposes: `vision/` (project direction and feature visions),
`strategic/` (architecture and feature catalog), `functional/` (devPNT-generated
functional snapshots), `reference/` (operative guides), `solutions/` (per-feature
analyses, discovery-by-grep), `audit/` (audit plan and handoff).
