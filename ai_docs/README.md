# ai_docs — reading guide

Must-reads for this repository, in order. The full manifest of canonical docs is
`INDEX.md` (generated — regenerate with `sdlc_check.py index`, never edit by hand).

1. `vision/project_vision.md` — why the skill exists (check its Status first).
2. `vision/roadmap_evoluzione_agenti.md` — the active evolution roadmap (v1.7.0 milestone: subagents, operative guides, devPNT seam).
3. `strategic/architecture.md` — how the package is built.
4. `audit/handoff.md` — where work stopped last session (if present).

Directory purposes: `vision/` (project direction and feature visions),
`strategic/` (architecture and feature catalog), `functional/` (devPNT-generated
functional snapshots), `reference/` (operative guides), `solutions/` (per-feature
analyses, discovery-by-grep), `audit/` (audit plan and handoff).
